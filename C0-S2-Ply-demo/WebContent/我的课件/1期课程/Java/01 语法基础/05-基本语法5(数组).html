数组

数组用于将相同类型的数据存储在连续的存储单元中
可以通过指定数组的名称和长度来声明数组
一旦声明数组的大小，就不能再修改
数组元素通过数组名和索引来访问，索引从0开始
数组可以是一维，也可以是多维

new关键字

在Java中，不允许将数组分配在栈内存中，只能分配在堆内存中
new是Java中内置的操作符
new操作符用来从堆中分配指定类型的内存空间，并返回相应类型的引用
首次创建数组时，所有值均初始化为0(int[]或double[])、false(boolean[])、null(引用类型)

关于数组长度

在Java中数组自带的length属性将负责保管数组的长度
在Java中如果试图访问不在有效索引以内的元素，将会引发数组越界错误

二维数组

值由行和列构成的, 我们称为二维数组或矩阵
在构造二维数组时，应确定所需要的行和列
例如:
   int rows=3;
   int columns=3;
   String[][] board = new String[ rows ][ columns ];

为访问一个特定的二维数组元素，应指定两个下标
例如:
   board[ 2 ][ 1 ] = “X”;

访问二维数组元素必须使用嵌套循环
