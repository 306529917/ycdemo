运算符

运算符是通过一定的运算规则操作一个或多个操作数，并生成结果的特定符号
运算符和操作数的有效组合称为{[表达式][方程式]}
Java提供了一系列丰富的运算符，包括
	算术运算符|关系运算符|逻辑运算符|条件运算符|赋值运算符


算术运算符

算术运算符主要用于数学计算，一般使用{[数值][字符]}作为操作数
下列{[][不]}属于一元运算符的是     |!+|-|++|--|!%
下列{[][不]}属于二元运算符的是     |+|-|!++|!--|*
取模( 求余数 )运算符是     |!+|!-|!/|%|!*

关系运算符

关系运算符可以测试两个操作数之间的关系（但不会改变操作数的值），关系表达式的结果为 boolean 类型
关系运算符{[包括][不包括]}:
|==|!=|>=|<=|>|<|@+=

关系运算符中判断大小关系{[包括][不包括]}:
|@==|@!=|>=|<=|>|<

可以判断两个操作数可能相等的关系运算符{[不包括][包括]}:
|@==|@!=|@>=|@<=|>|<

结果为 true 表示一定不相等的关系运算符{[包括][不包括]}:
|==|@!=|>=|<=|@>|@<


逻辑运算符

逻辑运算符{[必须][不一定非要]}与 boolean 型操作数一起使用
下列{[][不]}属于二元运算符的是:
@!  #  &  #  &&  #  |  #  ||  #  

{[][不能]}实现逻辑"与"的运算符是:
@!  #  &  #  &&  #  @| #  @||  #  

{[][不能]}实现逻辑"或"的运算符是:
@!  #  @&  #  @&&  #  |  #  ||  #  

{[][不能]}实现逻辑"非"的运算符是:
!  #  @&  #  @&&  #  @|  #  @||  #  

	
短路与和短路或
对于短路与（&&），如果其左侧为{[false][true]}，则不再计算其右侧，整个表达式的结果为false
对于短路与（&&），如果其左侧为{[true][false]}，时才计算右侧
对于短路与（||），如果其左侧为{[true][false]}，则不再计算其右侧，整个表达式的结果为true
对于短路与（||），如果其左侧为{[false][true]}，时才计算右侧

赋值运算符

赋值运算符的赋值方向是{[从右至左的][从左至右的]}
下列哪个赋值运算符{[不][]}会进行数学计算:
/= | +=  | != | *= | %=


条件运算符

条件运算符又叫三元运算符, 它的一般形式是：表达式1 ? 表达式2 : 表达式3
如果 表达式1 的结果为true，则执行{[表达式2][表达式3]}
如果 表达式1 的结果为false，则执行{[表达式3][表达式2]}

运算符的优先级

一个表达式由多个运算符组成，优先级决定运算符的计算顺序


	
	m 的值是|!15|!50|55|!5



	
	k 的值是|!5|10|!8|!9



	
	x 的值是|!5|!20|!15|10



	
	b 的值是|!5|3|!15|!4



	
	m 的值是|!6|!7|!8|9
