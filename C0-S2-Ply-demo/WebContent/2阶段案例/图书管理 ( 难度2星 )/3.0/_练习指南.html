实现功能（在2.0基础之上实现的扩展功能）

  1. 图书表新增 bookimage 字段，用于保存图书的图片路径（数据库脚本.sql）；
  2. 给图书表现有的图书记录设置了图片路径（数据库脚本.sql）；
  3. 图书列表加入图书图片的展示；
  4. 在保存图书信息的同时，将图片文件文件上传到 upload 目录下；

知识点

  1. 文件上传方式的表单提交；
  2. SmartUpload 实现文件上传；
  3. 使用 Java 程序创建文件夹；

相关 JAR 包支持

  1. jsmartcom_zh_CN.jar

难点

  1. SmartUpload 上传组件的使用；
  2. 文件上传方式的表单提交，与普通方式表单提交的设置上的区别；
  3. 上传的文件保存在哪里？
  4. 文件上传控件（input type='file'）与图片控件（img）的配合使用
